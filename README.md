BrightTV Viewership Analytics Project
Overview
This repository contains an analytical case study focused on BrightTV, a subscription-based streaming platform. The objective of the project is to generate actionable insights that support the Customer Value Management (CVM) team in growing the platform’s subscription base.
The analysis leverages user profile data and viewership transaction data to uncover trends, understand consumption behavior, and recommend strategies for user growth and engagement.
________________________________________
Business Objective
BrightTV’s CEO has set a goal to increase the subscription base within the current financial year.
This project aims to:
•	Analyze user behavior and viewing patterns
•	Identify key drivers of content consumption
•	Recommend content and strategies to boost engagement
•	Propose initiatives to grow the user base

Dataset Description
The dataset includes:
•	User Profiles: Demographic and account-related information
•	Viewer Transactions: Session-level viewing data
Key Notes:
•	Each record represents one viewing session per subscriber
•	Date and time fields are provided in UTC and must be converted to South African Standard Time (SAST)
•	Additional derived features may be created to enhance analysis (e.g., day of week, time of day, session duration)
